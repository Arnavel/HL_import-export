#Экспорт/Импорт хайлоад блоков  
К сожалению битрикс не умеет выгружать хайлоады по-этому пришлось писать свой скрипт)  

##Экспорт хайлоада  
1. Загрузить скрипт на сайт  
2. Ввести id хайлоада в соответствующую формочку и засабмитить   
3. Скопировать ссылку на получившийся json файл  

##Импорт хайлоада 
1. Загрузить скрипт на нужный сайт  
2. Вставить ссылку на файл и засабмитить  
3. Наслаждаться результатом  

Если в Хайлоаде есть свойство типа файл, то будут сформированы ссылки на эти файлы.   
Скрипт импорта загрузит файлы с удаленнного сервера и сохранит в битриксе
